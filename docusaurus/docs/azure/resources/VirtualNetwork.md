Provides a virtual network.

```js
const virtualNetwork = await provider.makeVirtualNetwork({
  name: `virtual-network`,
  dependencies: { resourceGroup },
  properties: () => ({
    properties: {
      addressSpace: { addressPrefixes: ["10.0.0.0/16"] },
      subnets: [
        {
          name: `subnet`,
          properties: {
            addressPrefix: "10.0.0.0/24",
          },
        },
      ],
    },
  }),
});
```
