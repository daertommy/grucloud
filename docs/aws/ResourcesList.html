Resources List

List of resources for provider aws:

  • ACM: Certificate
  • APIGateway: Account, ApiKey, RestApi, Stage, Authorizer, UsagePlan
  • ApiGatewayV2: DomainName, Api, Stage, Authorizer, ApiMapping, Integration, Route, Deployment, VpcLink
  • AppRunner: Connection, Service
  • AppSync: GraphqlApi, DataSource, Resolver
  • AutoScaling: AutoScalingGroup, AutoScalingAttachment, LaunchConfiguration
  • CloudFormation: Stack
  • CloudFront: Distribution, CachePolicy, Function, OriginAccessIdentity
  • CloudTrail: Trail, EventDataStore
  • CloudWatch: MetricAlarm
  • CloudWatchEvents: ApiDestination, Connection, EventBus, Rule, Target
  • CloudWatchLogs: LogGroup
  • CodeBuild: Project
  • CodeDeploy: Application, DeploymentGroup
  • CodePipeline: Pipeline
  • CodeStarConnections: Connection
  • CognitoIdentityServiceProvider: IdentityProvider, UserPool, UserPoolClient, UserPoolDomain
  • DynamoDB: Table, KinesisStreamingDestination
  • EC2: CustomerGateway, DhcpOptions, DhcpOptionsAssociation, FlowLogs, Ipam, IpamScope, IpamPool, IpamPoolCidr, KeyPair, NetworkInterface, Volume, VolumeAttachment, Vpc, InternetGateway, InternetGatewayAttachment, EgressOnlyInternetGateway, NatGateway, Subnet, RouteTable, RouteTableAssociation, Route, SecurityGroup, SecurityGroupRuleIngress, SecurityGroupRuleEgress, ElasticIpAddress, ElasticIpAddressAssociation, Instance, LaunchTemplate, NetworkAcl, ManagedPrefixList, PlacementGroup, VpcEndpoint, VpcPeeringConnection, VpcPeeringConnectionAccepter, TransitGateway, TransitGatewayRoute, TransitGatewayRouteTable, TransitGatewayPeeringAttachment, TransitGatewayVpcAttachment, TransitGatewayRouteTableAssociation, TransitGatewayRouteTablePropagation, VpnGateway, VpnConnection
  • ECR: Repository, Registry
  • ECS: CapacityProvider, Cluster, TaskDefinition, Service, TaskSet, Task, ContainerInstance
  • EFS: FileSystem, AccessPoint, MountTarget
  • EKS: Cluster, NodeGroup
  • ElasticLoadBalancingV2: LoadBalancer, TargetGroup, Listener, Rule
  • Glue: Job
  • IAM: OpenIDConnectProvider, User, Group, Role, Policy, InstanceProfile
  • Kinesis: Stream
  • KMS: Key
  • Lambda: Layer, Function, EventSourceMapping
  • NetworkFirewall: Firewall, Policy, RuleGroup, LoggingConfiguration
  • NetworkManager: GlobalNetwork, CoreNetwork, Site, Device, Link, TransitGatewayRegistration
  • Organisations: Root, Organisation, OrganisationalUnit
  • RAM: ResourceShare, PrincipalAssociation, ResourceAssociation
  • RDS: DBProxy, DBProxyTargetGroup, DBSubnetGroup, DBCluster, DBInstance
  • Route53: HealthCheck, HostedZone, ZoneVpcAssociation, VpcAssociationAuthorization, Record
  • Route53Domains: Domain
  • Route53RecoveryControlConfig: Cluster, ControlPanel, RoutingControl, SafetyRule
  • Route53RecoveryReadiness: Cell, ReadinessCheck, RecoveryGroup, ResourceSet
  • Route53Resolver: Endpoint, Rule, RuleAssociation
  • S3: Bucket, Object
  • SecretsManager: Secret
  • SNS: Topic, Subscription
  • SQS: Queue
  • SSM: Document, Parameter
  • StepFunctions: StateMachine
  • WAFv2: WebACL, WebACLCloudFront, WebACLAssociation
