Key

Provides a Key from the KeyVault group

Examples​

Create a key​

exports.createResources = () => [
  {
    type: "Key",
    group: "KeyVault",
    name: "myKey",
    properties: () => ({ properties: { kty: "RSA" } }),
    dependencies: ({}) => ({
      resourceGroup: "myResourceGroup",
      vault: "myVault",
    }),
  },
];

Dependencies​

  • ResourceGroup
  • Vault
