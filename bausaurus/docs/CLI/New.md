---
id: New
title: gc new
---

The **new** commands create an empty project for AWS, Azure and GCP.

## Command Options

```
gc help new
```

```
Usage: gc new [options]

Create a new project

Options:
  -p, --profile <string>  AWS only: The AWS profile (default: "default")
  -h, --help              display help for command
```
